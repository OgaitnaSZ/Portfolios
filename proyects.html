Aplicación Web de Gestión Médica

Desarrollé una aplicación web para la gestión integral de pacientes en centros médicos, utilizando Angular para el frontend y PHP + MySQL para el backend, enfocada en optimizar el manejo de historias clínicas, consultas y documentación médica.

Entre sus principales funcionalidades se encuentran:

  • Gestión completa de pacientes: alta, modificación, eliminación y búsqueda eficiente.
  • Visualización de la historia clínica individual de cada paciente, organizada por consultas.
  • Registro de consultas médicas con fecha, descripción detallada y posibilidad de adjuntar archivos (como análisis clínicos, estudios o informes complementarios).
  • Dashboard centralizado con un resumen general de pacientes, consultas recientes y estadísticas clave.
  • Importación masiva de pacientes mediante la carga de archivos Excel, facilitando la migración de datos o la incorporación de nuevos registros.

La plataforma está diseñada para ser ágil, segura y escalable, ofreciendo a los profesionales de la salud una herramienta intuitiva para la administración eficiente de sus pacientes y sus historiales médicos.

Plataforma de Reservas de Alojamientos

Desarrollé una aplicación web similar a Booking, utilizando Angular para el frontend, y PHP + MySQL para el backend, orientada a facilitar la búsqueda, publicación y reserva de alojamientos y experiencias turísticas en múltiples ciudades.

Entre sus principales funcionalidades se encuentran:

  • Publicación y gestión de alojamientos, con personalización de características por habitación y propiedad.
  • Sistema de reservas para habitaciones y actividades.
  • Destacados en portada, permitiendo a los anfitriones destacar sus alojamientos.
  • Pasarela de pagos integrada con MercadoPago.
  • Sistema de autenticación/login para usuarios y anfitriones.
  • Historial de reservas y visitas anteriores.
  • Sistema de reseñas, disponible una vez finalizada la estadía o actividad.

La plataforma está pensada para ser intuitiva, segura y escalable, ofreciendo una experiencia fluida tanto para viajeros como para anfitriones.

KanbanApp – Organización y Gestión de Tareas para Equipos

Esta es una aplicación web de gestión de tareas tipo Kanban, similar a Trello, desarrollada con ASP.NET Core, utilizando Razor Pages, Bootstrap y Javascript para la interfaz de usuario. La aplicación permite a los usuarios gestionar tableros, tareas y equipos con diferentes niveles de acceso. Cuenta con autenticación de usuarios mediante cookies de sesión y variables de sesión.

Caracteristicas Principales:

  • Autenticación y control de acceso: Inicio de sesión con roles de usuario.
  • Gestión de tableros: Crear, modificar y listar tableros con tareas asignadas.
  • Gestión de tareas: Flujo de tareas basado en estados: Ideas, ToDo, Doing, Review, Done.
  • Gestión de equipo: Visualización de usuarios y sus actividades.
  • Manejo de errores: Redirección de páginas inexistentes y logging de errores con ILogger.

Gestor de Clientes para Aseguradora

Desarrollé una aplicación web completa utilizando Angular para el frontend y Firebase como backend, orientada a optimizar la gestión de clientes en una aseguradora.

La aplicación permite registrar, editar y eliminar clientes, así como gestionar ventas asociadas a cada uno. Incorpora un sistema integral de administración de cuotas, que facilita el seguimiento detallado de pagos: cuotas pagadas, pendientes y vencidas, todo en una interfaz intuitiva y segura.

Esta herramienta mejora significativamente la organización interna y el control financiero de la aseguradora, ofreciendo una solución escalable y en tiempo real.
